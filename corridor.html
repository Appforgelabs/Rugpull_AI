Edit ticker

All values stay on your machine. Nothing is sent anywhere unless you use the FMP fetch.
CSV: label, reportDate(YYYY-MM-DD), eps, projected(0/1) — one quarter per line. Paste straight from a spreadsheet.
CSV: date(YYYY-MM-DD), close — daily or weekly. Only the last "Sentiment" days drive the multiple; the rest just draws the price line.
Used only if you have fewer than 4 forward quarters — extends EPS via EPS×(1+g/4)ⁿ so NTM can always be computed.

Auto-fill from Financial Modeling Prep (optional)

Pulls daily prices, reported EPS, and forward analyst EPS estimates, then fills the boxes above for you to review before saving.
If the browser blocks the request (CORS), host this file (e.g. GitHub Pages) or route through a tiny proxy. See the companion guide for details.

Cloud sync — Google Apps Script

Store every ticker's data in a Google Sheet through an Apps Script web app, so the same numbers feed the model on any device — and you can edit EPS/prices right in the spreadsheet. The companion guide deploys the script in ~3 minutes.
Like the FMP fetch, cloud sync needs the page served over http(s) — it won't work from a file:// page. Deploy the script with access set to Anyone.
